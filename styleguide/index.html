what is caboodle?

Caboodle is a cross-browser collection of css animations that you can easily plugin and use around your site. It uses SCSS so you can quickly extend the animations you'd like without bogging down your HTML with classes. We've taken popular css animations from stud Front-End Developers like Animate CSS by Daniel Eden.
