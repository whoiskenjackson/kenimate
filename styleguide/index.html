what is caboodle?

Caboodle is a cross-browser collection of css animations that you can easily plugin and use around your site. We've taken popular css animations from stud Front-End Developers, like Animate CSS by Daniel Eden, and created a SCSS framework that you can use to easily plug and play around your site.
